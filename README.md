iOS上的简易天气预报
============
- 同时支持国家气象局与OpenWeather二种数据源。

- 调用gps自动定位到当前城市

- 查询热门城市 PM2.5值，数据取自 http://waqi.info/

空闲时间练手之作。

- Retina MacBook Pro
- Xcode 4.6
- iOS sdk 5.0
- iPhone4
